# CareerCanvas

CareerCanvas is a full-stack Resume Builder web application that allows users to create, edit, and download professional resumes easily.

---

## 📌 Features

- 📝 Create and edit resumes dynamically  
- 🎨 Multiple resume templates  
- 📄 Download resume as PDF  
- 🔐 User authentication (Login/Register)  
- 💾 Save and manage resumes  
- ⚡ Fast and responsive UI  

---

## 🛠️ Tech Stack

### Frontend
- HTML, CSS, JavaScript  


### Backend
- Node.js  
- Express.js  

### Database
- MongoDB
